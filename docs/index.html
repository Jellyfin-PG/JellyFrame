A Complete Customization Framework

The Sure Foundation
For Your Media Core

Built with absolute precision. JellyFrame provides a complete extension and theming framework for Jellyfin — replacing limitations with community-driven mods, themes, and server-side scripts, all managed from your dashboard.

01

Mods & Themes

Browse and install community mods and themes directly from the Jellyfin dashboard. CSS, browser scripts, and full server-side extensions — no file editing required.

02

Server Extensions

Write JavaScript that runs inside Jellyfin. Expose REST APIs, react to library events, schedule tasks, and communicate between mods — all with a clean, permissioned API.

03

Your Control

Self-hosted, open-source, and built on Jellyfin. Assets are cached on your server, permissions are explicit, and every extension runs under your terms — not theirs.
